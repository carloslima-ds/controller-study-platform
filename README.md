# Trilha Controller

Plataforma estratégica de gestão de estudos desenvolvida para estruturar minha jornada até Controller Estratégico de Alto Nível.

## Objetivo

Transformar o estudo de Controladoria em um sistema profissional, rastreável e mensurável, baseado em progresso real e domínio técnico.

## Funcionalidades

- Dashboard com evolução geral e por módulo
- Sistema Kanban (A Fazer, Em Estudo, Concluído)
- Registro de nível de domínio (0 a 10)
- Upload e visualização de PDFs
- Sistema de revisão espaçada
- Métricas de progresso
- Rank evolutivo

## Estrutura

Organizado em 7 módulos:

- Fundamentos Estruturais
- Performance & Análise Gerencial
- Controladoria Estratégica
- Riscos e Governança
- Executivo & Liderança
- Tecnologia & IA Aplicada
- Projetos Práticos

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
